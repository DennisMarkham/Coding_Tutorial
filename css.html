CSS

CSS makes things pretty.
Steve Jobs

CSS stands for Cascading Style Sheet. It is used to beautiful your page. To apply a CSS file to your HTML, you must put a link in the <head> tag of your HTML document, like so:

		<head>
		
		<link rel="stylesheet" type="text/css" href="mystyle.css">
		
		</head>

Just as HTML is built around tags, CSS is built around 'selectors'. CSS selectors are marked with { }. Here is an example of a CSS selector used to apply style to everything in a <p> tag:

		p
		{
		color:red	
		}

Style can also apply to elements of a certain class (using a .) or id (using a #).

		.favorites
		{
			background-color: blue
		}

		#aboutme
		{
			font-size: 200%
		}
