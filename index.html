Learning HTML is the first step to creating a webpage. While pretty much every webpage employs CSS, Javascript, and many other coding libraries, you can't have a webpage without HTML.

So what is HTML. HTML stand for Hyper Text Markup Language. What does that mean? I don't know.

What matters is that HTML provides the skeleton for your webpage. Other languages are analogous the to the skin and organs. HTML is a language written in "tags". Tags are letters or words contained in angular brackets ('><'). Some examples of html takes are <p>, <a>, and <li>.

Here is what HTML code might look like:

		<h1>About me</h1>
		

		<p>My name is Dennis Markham, I do not mark ham.  I am writing a paragraph here.</p>

Which would print out something akin to:

About me

;

My name is Dennis Markham, I do not mark ham. I am writing a paragraph here.

<h1> is heading tag. <p> is a paragraph tag. Tags can be placed inside one another. For example, the <li> tag is always supposed to go inside the <ol> or <ul> tags. For example:

		<p>Here are my favorite foods:</p>
		

		<ol>
		

		<li>bread</li>
		

		<li>yogurt </li>
		

		<li>pizza</li>
		

		</ol>

<ol> is the 'ordered list' tag. <ul> is the 'unordered list' tag. Hence, the code><ol> tag produces numbers, as you can see below.

Here are my favorite foods:

  1. bread
  2. yogurt
  3. pizza
<p>A paragraph tag. Creates a line break before and after.
<ol>Create an ordered list.
<ul>Create an unordered list.
